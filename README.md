# RISC-V Architecture: Design, Implementation, and Performance Analysis

This repository contains the ongoing research, assembly code, and architectural simulations for my Undergraduate Thesis at the Department of Physics (Electronics Direction).

## 🎯 Project Objective
The goal of this thesis is to explore the RISC-V Instruction Set Architecture (ISA) and its hardware implementation. The project focuses on:
- Comprehensive analysis of the RV32I base integer set.
- Visualization of the 5-stage pipeline datapath.
- Performance evaluation using architectural simulators (Ripes).

## 📁 Repository Structure
- **/docs**: Thesis manuscripts and documentation.
- **/src**: RISC-V Assembly source code and custom scripts.
- **/sim**: Simulation configurations and workspace files.
- **/media**: Architectural diagrams and performance graphs.

## 🛠 Tools & Environment
- **Simulator:** [Ripes](https://github.com/mortbopet/Ripes)
- **ISA:** RV32I (RISC-V 32-bit Integer)
- **Status:** Work in Progress 🚧
